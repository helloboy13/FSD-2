## Learning Outcomes

- **Understanding Authentication Mechanisms**  
  Learned the implementation and differences between Basic Authentication, Token-based Authentication, and JWT Authentication in a web application.

- **Developing Secure API Endpoints**  
  Gained experience in creating protected routes in Flask that require proper authentication before granting access.

- **Working with JSON Web Tokens (JWT)**  
  Learned how to generate, verify, and use JWT tokens to securely authenticate users and manage session-like behavior in APIs.

- **Handling HTTP Headers and Requests**  
  Practiced extracting credentials and tokens from HTTP headers such as `Authorization`, `x-auth-token`, and request bodies.

- **Building RESTful Authentication APIs using Flask**  
  Developed a functional authentication system using Flask, demonstrating how backend services handle user validation and access control.
